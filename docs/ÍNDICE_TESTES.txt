🚀 COMO COMEÇAR RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

PASSO 1: Abra 3 TERMINAIS DIFERENTES

Terminal 1 - EXPO SERVER:
$ npm start
(Deixe rodando, escolha opção "w" para web)

Terminal 2 - JSON SERVER (Banco de Dados):
$ npx json-server --watch bancoDeDados.json --port 3000
(Deixe rodando)

Terminal 3 - TESTES:
$ cd seu/projeto/aqui
$ npm install -D @playwright/test (se não tiver)

PASSO 2: Execute os Testes

# Todos de uma vez:
$ npx playwright test tests/login-tarefas.spec.ts

# Ou cada um individualmente:
$ npx playwright test tests/login-tarefas.spec.ts -g "Falha ao entrar"
$ npx playwright test tests/login-tarefas.spec.ts -g "Adicionar nova tarefa"
$ npx playwright test tests/login-tarefas.spec.ts -g "Editar tarefa"
$ npx playwright test tests/login-tarefas.spec.ts -g "Marcar tarefa"

PASSO 3: Veja os Resultados

$ npx playwright show-report
